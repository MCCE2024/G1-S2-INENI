⚡ Einführung: Energie-Management für KMU

Herausforderungen & Zielsetzung

Forschungslücke: Es existiert keine integrierte, praxisnahe Lösung, die Energieverbrauch, Smart-Meter-Daten und Echtzeit-Strompreise für KMU in einem Dashboard vereint.
Intransparenter
Energieverbrauch
Fehlende
Marktpreise
Ungenutztes
Potenzial
Problem: Klassische Monitoring-Lösungen sind für KMU oft zu komplex oder teuer – es fehlt ein zentrales Dashboard für Server-Energie und Marktpreise.
Integriertes Monitoring ermöglicht datenbasierte Entscheidungen für Kosten und Nachhaltigkeit.

📚 Stand der Forschung

Forschungsstand & Lücken

Thema Beschreibung
Prozessmessung in der Industrie Erfassung von Energie- und Prozessdaten in industriellen Anlagen
Smart Meter Integration Einbindung von intelligenten Stromzählern zur Verbrauchserfassung
Echtzeit-Datenübertragung Übermittlung von Messdaten in Echtzeit an zentrale Systeme
Gap-Analyse: Keine Lösung vereint Prozessmessung, Smart-Meter, Echtzeitpreise und Dashboard für KMU. Unser Ansatz schließt diese Lücke.

Dieses Projekt adressiert die Integrationslücke für KMU-Energie-Insights.

🗂️ Use Case: Echtzeit-Energieoptimierung

Praxisbeispiel & Nutzen

Methodik: Realtime-Datenerfassung, Cloud-Verarbeitung und reproduzierbare Messszenarien für KMU-Serverumgebungen.
Thema Beschreibung
On-Premises Serverbetrieb Lokaler Server für Mail, DNS und Filesharing im Unternehmen
Integration mit Cloud-Diensten Anbindung an Cloud-Services für Datenverarbeitung und Analyse
Beispiel: Ein KMU betreibt einen lokalen Server für Mail, DNS und Filesharing. Die Energieaufnahme wird über eine smarte Steckdose gemessen und mit aktuellen Strompreisen verknüpft.
Nutzen:
  • Transparenz über Energieverbrauch
  • Kostenoptimierung durch Marktpreis-Integration
  • Nachhaltigkeitsbewertung
Reale Anwendung für mehr Effizienz und Nachhaltigkeit.

🔗 Eingesetzte APIs

Schnittstellen für Datenintegration

Reproduzierbarkeit: Alle APIs und Messszenarien sind dokumentiert und ermöglichen eine akademische Replikation.
energyLIVE API
• Liefert Echtzeit-Verbrauchsdaten direkt vom Smart Meter.
• Visualisierung im energyLIVE Dashboard.
• Integration in eigene Systeme möglich.
API-Dokumentation energyLIVE
Strompreis API (smartENERGY)
• Bietet aktuelle und zukünftige Strompreise im 15-Minuten-Takt.
• Preise basieren auf der EPEX Spot Strombörse.
• Ermöglicht Kostenoptimierung durch dynamische Tarife.
API-Dokumentation Strompreis
Beide APIs werden kombiniert, um ein umfassendes Energiemonitoring und eine Kostenoptimierung zu ermöglichen.
Integration von Verbrauchs- und Preisdaten für maximale Transparenz

Zusammenfassung & Ausblick

Reproduzierbarkeit: Alle Messszenarien und APIs sind dokumentiert und können von Dritten nachvollzogen werden.
Ausblick: Weitere Optimierungsmöglichkeiten, Integration zusätzlicher Datenquellen und Automatisierung der Workload-Steuerung sind geplant.
